Javascript

By default Javascript is a synchronous language and a single threaded.

Execution context

Execute one line of code at a time. Each operation waits for the last operation to be executed.

Blocking code vs non-blocking code

Blocking code blocks the flow of the program. It read file synchronously. Non-blocking code doesn't block the program flow, read file Asynchronously. They both has their own adv and disadv according to the use case.
